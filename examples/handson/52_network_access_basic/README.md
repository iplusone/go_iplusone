# 52. Network Access Basic

`データアクセス > ネットワークアクセス` の入口です。

このハンズオンでは、外部データへアクセスする基本として、HTTP 経由でデータを取得する流れを学びます。

- `http.Get` で URL にアクセスする
- `Response` を受け取る
- `Body` から本文を読む
- `defer` で `Body.Close()` を必ず実行する
- 取得した `[]byte` を `string` として扱う

このリポジトリではネットワーク制限があるため、実際の外部サイトではなく `httptest` のローカルサーバーを使って学習します。考え方は実際の Web アクセスと同じです。

## 実行

```bash
go run ./examples/handson/52_network_access_basic
```

## 学習ポイント

1. ファイルアクセスと同様に、ネットワークアクセスでも `開く / 使う / 閉じる` が必要
2. `Response.Body` は `io.ReadCloser` なので、読み終わったら閉じる
3. 取得した本文はまず `[]byte` で受け取り、必要に応じて `string` に変換する
4. データアクセスはファイルだけでなく、ネットワークにも広がっていく
