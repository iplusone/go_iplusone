# 40 Interface Collections

このハンズオンでは、`interface` を配列に入れて、異なる構造体を同じ流れで処理する書き方を確認します。

## 学べること

- 同じ `interface` を実装した複数の構造体をまとめて扱える
- `SetValue` と `PrintData` のように、共通メソッドだけで処理を進められる
- 構造体ごとに中身や表示方法が違っても、`interface` 経由なら同じ型のように扱える

## 実行方法

```bash
go run ./examples/handson/40_interface_collections
```
