# 51 File Access Basic: ファイルもOSリソースだと意識して読む

このハンズオンでは、ファイルアクセスの基本を一通り確認します。`ReadFile` `WriteFile` だけでなく、`OpenFile` と `defer Close` まで入っているのでかなり実践的です。

前半では一時ディレクトリを作り、`os.WriteFile` で初期内容を書きます。次に `os.ReadFile` で読み直し、`[]byte` と `string` の関係も確認できます。

後半の `appendLine` が大事です。追記モードでファイルを開き、`defer f.Close()` で必ず閉じています。ここで、ファイルはただの文字列置き場ではなく「開いたら閉じる必要がある資源」だと見えてきます。

この回は、I/O の入口としてかなりよくできています。まずは全読み書き、その次に細かい制御へ進む流れが自然です。
