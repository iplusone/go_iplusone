# 34 New And Make: 似て見える2つの生成方法の役割を分けて覚える

このハンズオンでは `new` と `make` の違いを扱います。Go 初学者が一度は混乱しやすいところですが、役割で分けて考えるとかなり整理しやすいです。

`new` は「ゼロ値を持つ値を用意して、そのポインタを返す」ものです。一方 `make` は、スライス、map、channel のように内部構造の初期化が必要な型を使える状態にします。

だから `slice` `map` `channel` は `make`、単純な値のポインタが欲しいなら `new` という見え方になります。

この回では、何を作りたいのかに応じて生成方法が変わることを押さえれば十分です。文法の違いよりも、対象の型が何を必要としているかを見る回です。
