# 23 Variadic Functions: 引数の個数を固定しない関数を書く

このハンズオンでは、可変長引数を扱います。Go では `...` を使うことで、引数の個数が毎回違っても受け取れる関数を定義できます。

最初に見たいのは、「内部ではスライスとして受け取る」という点です。呼び出し側は複数の値を並べて書けますが、関数の中では `range` で回せる集まりとして扱えます。

引数が0個でも呼べること、既にあるスライスを `...` で展開して渡せることも重要です。後者は実務でもかなりよく出ます。

この回は、関数とスライスがつながる回でもあります。可変長引数は特別な魔法ではなく、スライスをベースにした便利な書き方だと理解するとかなり分かりやすいです。
