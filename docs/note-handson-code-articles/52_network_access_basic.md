# 52 Network Access Basic: HTTPで外のデータを取る流れを小さく試す

このハンズオンでは、`http.Get` を使ったネットワークアクセスの基本を学びます。外部サイトではなく `httptest` のローカルサーバーを使っているので、考え方に集中しやすい形です。

流れはとても実務的で、URL にアクセスし、`Response` を受け取り、`Body` を読み、最後に `Close` します。特に `Body` が `io.ReadCloser` で、`defer resp.Body.Close()` が必要な点は重要です。

取得した本文が `[]byte` で返ってくることも、この回で自然に確認できます。ネットワーク越しのテキストも、最終的にはバイト列として扱っているわけです。

この回は、HTTP アクセスの最小単位をかなりきれいに見せています。後の API や JSON の理解にも直結します。
