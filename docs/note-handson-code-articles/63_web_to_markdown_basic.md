# 63 Web To Markdown Basic: ここまでの知識をつないで1つの成果物にする

このハンズオンは、かなり気持ちのいい総まとめです。Web ページを取得し、HTML を Markdown へ変換し、最後にファイルへ保存します。

ここには `http.Get` に相当するネットワークアクセス、`io.ReadAll` による読み込み、外部ライブラリによる変換、`os.WriteFile` による保存が全部入っています。つまり、前のハンズオンで学んだことがひとつの流れとしてつながります。

さらに `httptest.NewServer` を使ってローカルで完結させているので、外部依存なしで考え方だけを追えるのも良いところです。

この回で見たいのは、HTML を Markdown に変換するライブラリそのものより、「データを取得して加工して保存する」という実装の流れです。ここまで来ると、Go で小さなツールを作る感覚がかなり出てきます。
