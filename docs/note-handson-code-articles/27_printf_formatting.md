# 27 Printf Formatting: 値を「どう見せるか」まで制御できるようになる

このハンズオンでは、`Printf` の書式指定をまとめて確認します。値を計算するだけでなく、見やすく出力する技術です。

`%d` `%f` `%s` `%q` `%v` `%T` が並ぶことで、数値、文字列、型情報などをどう表示し分けるかが見えます。特に `%T` は学習初期のデバッグにかなり便利です。

幅指定まで出てくるので、ただ値を出すだけでなく、整形して読みやすくする感覚も分かります。

この回は地味ですが、手元でコードを試すときの観察力を大きく上げてくれます。Go 学習では、`Printf` をうまく使えるだけで理解がかなり深まります。
