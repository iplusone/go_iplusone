# 44 Reflect Type Checks: 値を見て、その場で型を確かめる

このハンズオンでは `reflect.TypeOf` と `Kind` を使って、interface に入った値の型を調べます。型アサーションの前に「そもそも何が入っているか」を見る回です。

`reflect.TypeOf(value).Kind()` で、`int` なのか `string` なのかといった基本的な種類を判定できます。これを使うと、型が合うときだけ代入する流れを安全寄りに書けます。

サンプルでは、型が合わない値はゼロ値のままになります。ここで見たいのは、「reflect は万能変換ではなく、型を確認するための道具」だということです。

この回は少し抽象的ですが、動的な値を扱うときの考え方を広げてくれます。
