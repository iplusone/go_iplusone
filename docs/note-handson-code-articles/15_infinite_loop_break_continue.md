# 15 Infinite Loop Break Continue: ループをどう抜け、どう飛ばすか

このハンズオンでは、`for {}` による無限ループと、`break` `continue` の違いを確認します。ここを理解すると、ループ内の流れをかなり細かく制御できるようになります。

最初の無限ループでは、条件がない `for` がどう動くかを見ています。止める条件はループの中に書き、`break` で抜けます。Go ではこの形もかなり自然です。

次の `continue` の例では、特定の条件に当たったときだけその回の残りを飛ばして次へ進みます。`break` が「終わる」、`continue` が「その回だけ飛ばす」という違いをここで体で覚えるとよいです。

最後の例はその組み合わせです。奇数は飛ばし、偶数だけ処理し、ある条件でループ全体を止めています。

この回で大事なのは文法よりも、「ループのどの時点で何をしたいのか」を意識して読むことです。
