# 25 Higher Order Functions: 関数を受け取る関数、返す関数を体で覚える

このハンズオンでは高階関数を扱います。少し難しそうに聞こえますが、やっていることは「関数を引数に渡す」「関数を返す」の2つです。

変換処理や条件関数を渡せるようになると、同じ骨格の処理を再利用しつつ、中身だけ差し替えられるようになります。ここが高階関数の大きな価値です。

Go は関数型言語ではありませんが、こういう柔らかい書き方も自然にできます。だからこそ、単なる文法ではなく「振る舞いを渡している」と意識して読むと理解しやすいです。

この回は、処理を値として渡す感覚がつかめると一気に面白くなります。後のコールバックやミドルウェア理解にもつながります。
