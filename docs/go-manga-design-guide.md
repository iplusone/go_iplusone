# Go誕生秘話 デザインガイド v1

このドキュメントは、漫画シリーズ「Go誕生秘話」の共通デザインルールを固定するためのガイドです。

第2章以降の全話で、ここに書かれたレイアウト、配色、キャラクター、フッタ構成を基本ルールとして扱います。

## 目的

- 全話を並べたときにシリーズとしての統一感を出す
- 1話ごとの制作判断を減らす
- 技術漫画ポスターとして読みやすい型を固定する

## 1. 用紙

- A4縦型
- 技術漫画ポスター形式
- 上から下へ読む
- 1話 = 1枚

## 2. ヘッダ

高さ:

- 全体の10〜12%

背景:

- 濃紺 `#0F1B2D`

文字:

- 白

構成例:

- `第2章 Go革命`
- `第1話 継承を捨てる`

サブタイトル:

- `テーマ：`
- `複雑な継承の世界から、`
- `シンプルな未来へ`

## 3. 本文

背景:

- 白

構成要素:

- 大ゴマ
- 技術図解
- キャラクター解説
- 吹き出し

## 4. キャラクター

固定キャラクター:

- Rob Pike
- Ken Thompson
- Robert Griesemer
- Gopher

## 5. 技術図解

配色ルール:

- Go側: 水色 `#00ADD8`
- 従来技術: グレー
- 問題点: 赤
- 利点: 緑

## 6. フッタ

高さ:

- 全体の10〜12%

背景:

- 濃紺 `#0F1B2D`

構成:

- 左: 今日の一言
- 右: 次回予告

今日の一言 例:

```text
「シンプルであることは、
難しい。だからこそ価値がある。」

— Rob Pike
```

次回予告 例:

```text
第2話
compositionの思想
```

## 7. シリーズ統一ルール

全話共通で、以下の3層構成を固定する。

1. ヘッダ: 濃紺
2. 本文: 白
3. フッタ: 濃紺

これにより、第2章以降は雑誌連載のような統一感を持たせる。

## 運用メモ

- 各話ごとの個性は本文の構図と図解で出す
- ヘッダとフッタはできるだけ崩さない
- 技術テーマが変わっても図解の色ルールは維持する
- 100話規模で並べたときに一目で同シリーズと分かることを優先する
